# Bike Sales Analysis Using Excel

## Project Overview
This project analyzes customer purchasing behavior using Excel. The dataset is a publicly available practice dataset commonly used in data analytics tutorials.

## Objectives
- Analyze customer demographics and income levels
- Identify factors influencing bike purchases
- Build an interactive Excel dashboard

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers)
- Data Cleaning and Analysis

## Project Structure
- /Bike_Sales_Project
  - Bike_Sales_Analysis.xlsx
  - dashboard_screenshot.png

## Key Tasks
- Cleaned and standardized the dataset
- Created pivot tables for insights
- Built an interactive dashboard using slicers
- Interpreted results to support business decisions

## Key Insights
- Customers with higher income were more likely to purchase bikes
- Middle-aged customers showed the highest purchase rate
- Homeowners had higher purchase frequency

## Dataset Source
Public practice dataset popularized by Alex The Analyst (used for learning purposes).
